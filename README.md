# Diwan Dashboard — واجهة الديوان العام

تشغيل محلي سريع:

1. تثبيت التبعيات:

```bash
npm install
```

2. تشغيل السيرفر:

```bash
npm start
```

ثم افتح المتصفح على: http://localhost:3000

الملفات المهمة:
- [server.js](server.js)
- [public/index.html](public/index.html)

المرحلة الحالية: تم تنفيذ الواجهة الرئيسية فقط (Dashboard UI). انتظر تعليمات الربط بقاعدة البيانات والجداول وAPIs.
